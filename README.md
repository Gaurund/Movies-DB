# Movies-DB

## Описание

Movies-DB — это приложение для управления домашней фильмотеки. Программа позволяет сканировать подключаемые диски, добавлять, редактировать, удалять и просматривать информацию о фильмах с базовым графическим интерфейсом.

## Технологии

- **Python 3.13** — язык программирования
- **SQLAlchemy** — ORM для работы с базой данных
- **Tkinter** — встроенная библиотека Python для создания графического интерфейса

## Функциональность

- ✅ Сканирование подключаемых жёстких дисков и поиск на ней видео файлов
- ✅ Добавление фильмов в базу данных пакетным и ручным способами
- ✅ Редактирование информации о фильмах
- ✅ Удаление фильмов из базы
- ✅ Извлечение деталий о фильме из публично доступной информации в интернете
- ✅ Поиск и фильтрация по названию, году, жанру
- ✅ Просмотр подробной информации о фильме
- ✅ Сортировка по различным критериям
